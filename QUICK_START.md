# 快速开始指南

## 🚀 5分钟快速体验经济模组

### 第一步：基础设置
1. 启动带有经济模组的Minecraft服务器
2. 进入游戏世界
3. 输入 `/economy balance` 查看你的初始余额（100金币）

### 第二步：体验基础功能
```bash
# 查看帮助
/economy

# 查看余额
/economy balance

# 领取每日奖励
/economy daily
```

### 第三步：建造经济设施
1. **放置ATM方块**：右键查看自助服务菜单
2. **放置银行方块**：体验完整的银行服务
3. **放置商店方块**：了解商店管理系统

### 第四步：与朋友互动
```bash
# 转账给朋友（假设朋友名字是Steve）
/economy pay Steve 50

# 给朋友开具支票
/economy check 100 Steve
```

### 第五步：银行服务
```bash
# 存款到银行
/economy deposit 50

# 从银行取款
/economy withdraw 25

# 申请小额贷款
/economy loan 500
```

### 第六步：查看历史
```bash
# 查看交易历史
/economy history

# 查看服务器经济统计
/economy stats
```

## 🎯 常用场景

### 💰 日常使用
- 每天登录领取奖励：`/economy daily`
- 查看余额：`/economy balance`
- 转账给朋友：`/economy pay <玩家名> <金额>`

### 🏦 银行理财
- 存钱赚利息：`/economy deposit <金额>`
- 需要钱时取出：`/economy withdraw <金额>`
- 应急贷款：`/economy loan <金额>`

### 🏪 商业活动
- 与其他玩家交易
- 建立个人商店
- 参与拍卖活动

## ⚠️ 重要提示

1. **安全第一**：妥善保管你的财产，定期检查账户
2. **理性消费**：合理规划支出，避免过度负债
3. **积极参与**：多与其他玩家交易，体验经济系统的乐趣
